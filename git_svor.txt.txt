Git spurningar

1. Hvað gera eftirfarandi Linux skipanir?
	Cd		breyta skrá
	cd ~		Fer beint til baka (home directory)
	ls		listi af upplysingum
	pwd		Breytir um drif
	mkdir		Býr til skrá

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone	copyar geymslu sem er nu þegar til
	git log		Synir það sem þu hefur gert/sláð inn
	git status	synir hvar það er og hvort það virkar
	git diff	synir breytingar á skjalinu
	git checkout	upfærir skjalið þitt

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff	synir breytingar á skjali
	b) git diff –staged	Bretyir bara skjali í „staged“ svæði
	c) git diff commit1 commit2	sýnir mismunin á 2 akveðnum commit

4. 	Hvað er útgáfustýring (e.version control)? 
	Það synir allar utgafur af forritunu
